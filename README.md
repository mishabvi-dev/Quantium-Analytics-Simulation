## Quantium Data Analytics Virtual Internship - Task 1 🛒
This repository contains my analysis for the Quantium Virtual Internship. As a Data Analyst in the retail analytics team, I am analyzing chip purchasing behavior to provide strategic recommendations for the Category Manager.

### 🎯 Project Objective
To examine transaction and customer data, identify purchasing trends, and define customer segments to drive a half-year strategic plan for the chip category.

### 🛠️ Tech Stack & Skills
Language: Python

Libraries: Pandas (Data Wrangling), Matplotlib/Seaborn (Visualization), NumPy

Tools: Jupyter Notebook, Git/GitHub

Key Techniques: Data Cleaning, Outlier Detection, Feature Engineering, Customer Segmentation

### 📋 Task 1 Execution Checklist
[ ] Data Cleaning: Converting date formats and filtering out non-chip products (e.g., Salsa).

[ ] Outlier Removal: Identifying and removing bulk commercial transactions to prevent skewed metrics.

[ ] Feature Engineering: Extracting Pack Size and Brand Name from product descriptions.

[ ] Data Merging: Joining Transaction and Customer Behavior datasets.

[ ] Analysis: Segmenting sales by Lifestage and Premium Customer status.
